# 🌈 개인 홈페이지 마크업
영상제작자로 활동하던 때 제작한 포트폴리오용 홈페이지
- html과 css만을 이용한 마크업 작업
- 메인화면은 이력과 스킬 위주로 구성
- 포트폴리오 화면은 이미지 클릭 시 해당 영상 링크로 이동하도록 설정
